# Original Urdu/Punjabi Voice Transcript

## Transcript

“شادی کار بار شروع کیتا پر جے اونجڈانا ہووُس، چلوںا بیاں نہ توں تاں سدہبھی کرنا آ تودے ہونا دُوے جڑا ہویا کزناں نالو بھی جڈا ڈیوٹی والا بندہ تے پچھے بعد سارا کچھ جڈا او سیل مےن تڑڈر بک کر دی تے سیل ہون حوالے دے تے چھڑا ویتان اسین آنلائن سرچ چیک کر سکے اگر موبائل اپلیکیشن ہو سکدی تے اوہ ویگ نہ تے جے کوئی دوسری توسیع سمیٹ ڈاٹا بیس وغیرہ کچھ ہو سکدا تے اوہ تے سارا کچھ جڑا ہو سیل ہوندا اوہ نانڈی گالی اے بھائی اونج جد جد جانا ہو تے اوساں سیل کرنہی اوساں دا لال لال ڈیٹ ہووے تے لال لال چیزاں آنے تے چیزاں دیے”

## Interpreted Meaning

The client is saying that the business has started and they need a system where the duty person/order booker/salesman can book sales in the field. After sales are entered, the owner/admin should be able to check and search records online. A mobile application would be ideal, but if a web/database system can handle it, that is also acceptable. The system should show date-wise sales, item-wise sales, stock coming in, stock going out, and all sale records.

## Voice-note Confirmed Requirements

- Field salesman/order booker should enter sales.
- Owner/admin should be able to check/search online.
- Mobile app is preferred.
- Web/database system is acceptable.
- Date-wise sales record is important.
- Item-wise sales and stock movement are important.
- Duty/order-booker workflow is essential.
